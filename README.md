# Project Title

## Description
This project is an application to manage tasks and track streaks, allowing users to effectively add and display tasks with streaks of completion.

## Setup Instructions
1. Clone the repository: `git clone https://github.com/RyTheGuy18/streak_master.git`
2. Navigate to the directory: `cd streak_master`
3. Install dependencies: `npm install`
4. Start the application: `npm start` to run in development mode.
